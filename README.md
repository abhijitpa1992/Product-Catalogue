# Product Catalogue
This project is created for assignment submission purpose in Wishing Step Pictures.
A table view with product catalogue.

## Features

Offline json
UITableView
Lazy Loading Haldle
NavigationViewController
WKWebKit

## Supported Devices

iPhone with iOS 10 or later
iPad with iOS 10 or later

## Installing

Clone the project.
Go to application folder from terminal
Run the comman 'pod install' (requires CocoaPods)
open Product Catalogue.xcworkspace.
